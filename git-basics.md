# Gitの基本操作

## Gitとは

Gitは、分散バージョン管理システムです。ファイルの変更履歴を管理し、複数のバージョンを効率的に管理できます。

## 基本的なコマンド

### リポジトリの初期化

```bash
git init
```

現在のディレクトリをGitリポジトリとして初期化します。

### ファイルの状態確認

```bash
git status
```

現在のリポジトリの状態を確認します。変更されたファイルやステージングエリアの状態が表示されます。

### ファイルをステージングエリアに追加

```bash
git add <ファイル名>
git add .  # すべてのファイルを追加
```

変更したファイルをステージングエリアに追加します。

### コミット

```bash
git commit -m "コミットメッセージ"
```

ステージングエリアの変更をコミットします。コミットメッセージは変更内容を簡潔に説明します。

### 履歴の確認

```bash
git log
git log --oneline  # 1行で表示
```

コミット履歴を確認します。

### ブランチの作成

```bash
git branch <ブランチ名>
```

新しいブランチを作成します。

### ブランチの切り替え

```bash
git checkout <ブランチ名>
git switch <ブランチ名>  # 新しいコマンド
```

別のブランチに切り替えます。

### ブランチの一覧

```bash
git branch
```

すべてのブランチを表示します。

### マージ

```bash
git merge <ブランチ名>
```

指定したブランチを現在のブランチにマージします。
